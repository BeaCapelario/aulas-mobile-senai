Este repositório reúne os projetos e exercícios desenvolvidos durante as aulas de desenvolvimento mobile no SENAI. O foco das atividades foi o aprendizado de tecnologias cross-platform para a criação de aplicativos móveis modernos, explorando desde componentes básicos até a integração com recursos nativos do dispositivo.

---

### O que foi explorado:
* Componentes Nativos: Uso de View, Text, Image e ScrollView para construção de interfaces mobile.
* Flexbox Mobile: Aplicação de layouts flexíveis adaptados para diferentes tamanhos de tela e orientações.
* Hooks e Estados: Gerenciamento de dados e ciclo de vida de componentes dentro do ambiente mobile.
* Navegação: Implementação de fluxos entre telas utilizando bibliotecas de navegação (Stack, Tab).
* Expo Ecosystem: Utilização do ecossistema Expo para agilizar o desenvolvimento, testes e acesso a APIs nativas.

---

### Tecnologias Utilizadas
| Tecnologia | Descrição |
| :--- | :--- |
| React Native | Framework para desenvolvimento de apps nativos usando React. |
| Expo | Conjunto de ferramentas e serviços para projetos React Native. |
| JavaScript | Linguagem base para o desenvolvimento da lógica do app. |
| Node.js | Ambiente de execução e gerenciamento de dependências. |
